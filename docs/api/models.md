# Data Models

The following is a list of the different data models in this app.


| Model | Purpose                                                                                                                                                                                |
|-------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Tab** | This model is the music tabular sheet itself. It contains the file key to retrieve the document from the bucket and also the metadata for the file such as artist name, song name etc. |
